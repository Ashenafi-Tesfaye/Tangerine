Subtask 6: Dictation - Letter Writing

I will say the name of a letter; you should listen carefully. After I have told you the name of the letter, I want you to write the letter on this paper. You can write the letter in either capital or lower case.

Let’s practice: The letter I want you to write is “s” as in “sama”. Write the letter “s”.

[If the child writes the letter correctly, say:] Good, that is how you write the letter “s.”

[If the child does not respond correctly, say:] The letter “s” is written like this: [write letter for child].  

Okay, listen and let’s get started.
